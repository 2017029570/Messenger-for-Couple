캘린더(FSCalendar)와 데이터베이스(FirebaseDatabase) API 를 이용하여 해당 날짜에 대한 커플의 공유 일정을 확인, 추가 및 삭제할수 있는 기능이 들어간 탭 입니다. 먼저 일정을 확인하고 싶은 경우 캘린더의 해당 날짜를 누르면 아래의 TableView에 일정들이 나오는 것을 알수 있으며 일정을 추가하거나 삭제하고 싶은경우 상단의 추가 삭제 버튼을 눌러 수정하고자 하는 일정을 입력하고 Complete 버튼을 누르면 아래의 TableView에 추가되고 사용자의 연인에게도 그 일정이 수정되어 나타나도록 기능을 넣었습니다.

다른 커플 메신저앱과 가장 차별화된 부분입니다. 우선 진짜 사랑한 날짜를 분석하기 위해 커플은 매일 자가투표를 하게 됩니다. 본인이 생각했을 때 진짜 사랑했는지 투표합니다. 평일동안에는 저녁9시부터 새벽3시까지만 투표 가능합니다. 투표는 하루에 한 번만 가능하고 만약 투표를 하지 않으면 사랑하지 않은 것으로 간주합니다. 매주 일요일에 그 주에 대한 사랑의 평가를 알려줍니다. 우선 자가투표는 양쪽 모두 사랑했다고 선택했을 때에만 사랑했다고 평가합니다. 그리고 그 주에 몇 번 만났는지를 캘린더에 작성한 일정의 개수로 평가합니다.
